Overview of all electronic devices¶

This is a list of all electronic devices for which we have drivers available. Most devices inherit from the instrument class, while others do not.

Rigol_DS1054.py¶

File location: stlab/devices/Rigol_DS1054.py.

Imported via autodetect:

import stlab
dev = stlab.adi(addr)

Module for instance of a Rigol oscilloscope

This module contains the functions necessary to control and read data from a Rigol oscilloscope. The programming guide from Rigol can be found at http://int.rigol.com/File/TechDoc/20151218/MSO1000Z&DS1000Z_ProgrammingGuide_EN.pdf The module has been tested with a DS1054Z, but should also work with the entire MSO1000Z/DS1000Z series.

Reads what is on screen at low resolution

Given an input channel, this function reads the data back with a resolution of 1200 points. Note that this is a fast operation, but you will lose quite a significant amount of data because the precision is range/1200

ReadWaveData(n=1, mdep=6000000.0)[source]¶

Reads what is on screen at high resolution

Given an input channel, this function reads the data back with a resolution limited by the memory depth and bandwidth limit. Note that this is a high resolution but relatively slow measurement; per channel the data retrieving takes about 31 seconds.

Parameters:
